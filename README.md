
# Abhiram Gundekari's Portfolio

This is your professional portfolio built with Next.js, Tailwind CSS, and Genkit AI.

## 🚀 How to Add Your Files

To make your resume and profile picture appear on the site, follow these exact steps:

### 1. Prepare your files
- **Profile Picture:** Find your photo and name it exactly `profile.jpg` (all lowercase, must be a .jpg file).
- **Resume:** Find your resume PDF and name it exactly `resume.pdf` (all lowercase).

### 2. Upload to the `public/` folder
In the file explorer on the left side of this screen:
1.  Locate the folder named `public`.
2.  **Right-click** on the `public` folder.
3.  Select **Upload**.
4.  Choose your `profile.jpg` and `resume.pdf` files.
5.  Wait for the upload to complete.

### 3. Verify
- Refresh the preview. 
- Your photo should now appear in the circular frame at the top.
- The "Download Resume" buttons should now successfully download your PDF.

### Troubleshooting
- **Image not appearing?** Ensure the filename is `profile.jpg`. If it is a `.png`, rename the actual file on your computer to `.jpg` before uploading, or ask me to change the code to support `.png`.
- **Resume not downloading?** Ensure the filename is exactly `resume.pdf`.
