# **App Name**: PortfolioPro

## Core Features:

- Skill Showcasing: Visually highlight my Python, SQL, and DSA skills.
- Academic Details: Clearly display academic background and achievements.
- Project Presentation: Showcase projects with descriptions, code samples, and links to GitHub repositories.
- Resume Integration: Include a dedicated resume section with a downloadable resume option.
- Contact Information: Provide contact details (email, phone) with active links to LinkedIn and GitHub profiles. The links are to be read as a tool and included only if they appear relevant given the user's prompt.
- GitHub Contribution Tracking: Fetch and display recent GitHub contributions.
- Design Enhancement: The new portfolio redesign will have clean design and improved user experience.

## Style Guidelines:

- Primary color: Deep Blue (#2E5266) to convey professionalism and trustworthiness.
- Background color: Light Grey (#F0F4F7), providing a clean and modern backdrop.
- Accent color: Teal (#26A69A), used to highlight skills, project titles, and key information, drawing attention and complementing the primary blue.
- Body text: 'Inter', sans-serif, known for its legibility and clean design, ensuring a pleasant reading experience. Use for all body text and UI elements.
- Headlines: 'Space Grotesk', sans-serif, use for all main headings and subheadings.
- Use simple, professional icons to represent skills, technologies, and sections.
- Maintain a clean, well-structured layout with clear sections and intuitive navigation.
- Subtle animations when hovering over projects or skills to provide interactivity and engage users.
